Detailed Description

Provides material properties for steel.

This namespace contains lookup tables and helper functions to retrieve temperature-dependent thermodynamic properties of steel, specifically:

  • Specific Heat Capacity (Cp)
  • Density (rho)
  • Thermal Conductivity (k)

All functions accept temperature in Kelvin [K] and return values in standard SI units unless otherwise specified.
